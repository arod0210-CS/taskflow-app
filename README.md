TaskFlow App 📝🚀

TaskFlow is a simple and interactive task management web app designed to help users stay organized, track progress, and stay motivated with built-in gamification features.

🔗 Live Demo: https://arod0210-cs.github.io/taskflow-app/

📌 Project Overview

This project was built as part of my Computer Science coursework and personal portfolio development. The goal was to create a functional and visually engaging task tracker using core front-end technologies while improving my skills in JavaScript logic, DOM manipulation, and user experience design.

⚙️ Features Implemented
✅ Add, edit, and delete tasks
✅ Mark tasks as complete
✅ Task priority levels (Low, Medium, High)
✅ Search and filter tasks
✅ Drag-and-drop task reordering
✅ Dark mode toggle
✅ Multiple theme options (including custom theme)
✅ Local storage persistence (tasks saved in browser)
🎮 Gamification System
⭐ XP system for completing tasks
💰 Coin tracking
🔥 Streak counter
🎯 Challenge tracking
🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
LocalStorage API
GitHub Pages (for deployment)
🧠 What I Learned
How to manage application state using LocalStorage
DOM manipulation and event handling in JavaScript
Building interactive UI features (drag-and-drop, modals, filters)
Structuring a small front-end project from scratch
Improving user experience with themes and visual feedback
🚧 Work in Progress / Future Improvements
